# control flow
## if statement

```properties
if (condition) {
    ## do stuff
}

## example

my_num := 8

if (my_num < 68) {
    std_out("value is smaller than 68")
}
```

## if else

```properties
string_var := "some text"

if (string_var == "different text") {
    ## do stuff
}
else {
    ## do something different
}
```
